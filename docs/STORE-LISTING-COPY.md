## Description (mandatory)

Scalpel is a fast, local PDF editor for Windows. Open a PDF and view, annotate, edit, sign, and organize it — all on your own machine. No account, no cloud, no telemetry.

Scalpel does everything in one lightweight app:

• View — smooth single-page, continuous, two-page, and grid views with quick zoom and full-text search.
• Annotate — highlight text, draw ink, and add text notes anywhere on the page.
• Edit text — change existing text directly in the document.
• Organize pages — merge, split, reorder, rotate, and delete pages.
• Sign — draw or reuse a saved signature and place it where you need it.
• Fill forms — complete PDF forms and save them.
• Flatten — burn annotations and form fields into the page for a final, locked copy.
• Print — print directly from the app.

Privacy by design: every file is processed entirely on your device. Scalpel works fully offline, never uploads your documents, and includes no analytics, ads, or accounts.

Scalpel is free and open-source software (GPLv3). Source code is available on GitHub.

## Notes for certification (Submission options → paste here)

Scalpel is a free, local-only PDF editor. No login, account, or credentials are required — open the app and use it immediately. It works fully offline; no network connection is needed and nothing is uploaded. There are no in-app purchases and no telemetry. To test: launch the app, open any PDF (or use File ▸ Open), then try annotating, editing text, reordering pages, and saving. The app is GPLv3 open-source; full source is bundled with each release and at https://github.com/blakazulu/ScalpelPDF. The only declared capability is runFullTrust, standard for a packaged Win32 desktop app.
