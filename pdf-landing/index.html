What it does

View & Navigate

High-quality PDFium rendering. Four view modes - single page, continuous scroll, two-page, and a grid of every page - plus thumbnails, Ctrl+scroll zoom, and keyboard shortcuts.

Edit Text

Double-click any line of text to edit it inline. Font matching from the original PDF preserves the look of the document.

Annotate

Text boxes, freehand drawing, and highlight overlays with customizable colors, sizes, and opacity. Undo and clear per page.

Signatures & Images

Draw reusable signatures or import a PNG/JPG/BMP. Drop images onto any page as resizable annotations, burned into the PDF on save.

Merge & Split

Combine multiple PDFs into one or extract selected pages into a new file. Drag-and-drop page reordering, rotate, and delete in the sidebar.

Print + Preview

A built-in print dialog with a real page preview - pick printer, orientation, copies, and page ranges. Annotations are flattened into the output.

Flatten PDF

Save a fully uneditable version by rasterizing every page at 150 DPI via PDFium, now across multiple CPU cores for speed. Annotations burned in first.

Search & Select

Ctrl+F full-text search with highlighted results. Drag-select to copy text, or Ctrl+A to grab the whole page.

Forms & Bookmarks

Fill interactive PDF forms - text fields, checkboxes, radio buttons - and save back. Outline/bookmark navigation in the sidebar.

Six Themes

Dark, Light, High Contrast, Blood, Greed, and Cyanotic, switchable live in Settings. The whole app, including this site, follows.

Localized

UI translated into English, Spanish, and Traditional Chinese, selectable in Settings. Contribute a translation on GitHub.

Install or Run Portable

Drop the EXE anywhere and run it (USB, network share, Downloads), use the built-in installer, or winget install scalpel. No admin rights needed.
